Second Project
This project has demonstrations of basic HTML and CSS attributes.
It has the following requirement: 

			1: My portfolio should have a welcome section.
			2: The welcome section should have an h1 element that contains text.
			3: My portfolio should have a projects section.
			4: The projects section should contain at least one element with a class of project-tile to hold a project.
			5: The projects section should contain at least one link to a project.
			6: My portfolio should have a navbar.
			7: The navbar should contain at least one link that I can click on to navigate to different sections of the page.
			8: My portfolio should have a link with an id of profile-link, which opens my GitHub or FCC profile in a new tab.
			9: The height of the welcome section should be equal to the height of the viewport.	
			10: The navbar should always be at the top of the viewport.
